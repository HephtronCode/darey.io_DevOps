## About This Repository

This repository is a curated portfolio of DevOps, AWS Cloud, and Linux projects completed as part of the darey.io DevOps learning journey. Each project demonstrates practical skills in cloud infrastructure, automation, shell scripting, security, and system administration. The projects are organized by topic and complexity, with detailed documentation and screenshots for easy learning and reference.

## How to Use This Repository

1. **Browse Projects:** Each folder contains a README with step-by-step instructions, screenshots, and code samples.
2. **Reproduce Labs:** Follow the guides to practice AWS, Linux, and DevOps skills hands-on.
3. **Learn by Example:** Study shell scripts, Terraform configs, and automation patterns.
4. **Adapt for Your Needs:** Use and modify scripts for your own cloud or DevOps projects.

## Project Details

### AutoScaling-with-ALB

- **Goal:** Learn AWS Auto Scaling using Launch Templates and integrate with ALB.
- **Skills:** EC2, Launch Templates, Auto Scaling Groups, Scaling Policies, Load Balancers.
- **Highlights:** Stepwise setup, scaling policy configuration, ALB integration, demand simulation.

### AWS_Account-Creation

- **Goal:** Guide to creating and configuring a new AWS account.
- **Skills:** Cloud fundamentals, AWS onboarding, security best practices.
- **Highlights:** Screenshots, step-by-step account setup, root/IAM user distinction.

### AWS_S3-Project

- **Goal:** Master AWS S3 for file storage and management.
- **Skills:** S3 bucket creation, file upload/download, access control.
- **Highlights:** Visual walkthrough, unique bucket naming, versioning, ACLs.

### AWS-resourceCreation-shellScripting

- **Goal:** Automate AWS resource creation with shell scripts and AWS CLI.
- **Skills:** Shell scripting, AWS CLI, EC2, S3, key pairs.
- **Highlights:** Provisioning scripts, permissions, business unit automation.

### AWS-Vpc-Project

- **Goal:** Build a secure, multi-tier AWS VPC from scratch.
- **Skills:** VPC, subnets, gateways, security groups, EC2, routing.
- **Highlights:** Visual guides, CIDR planning, public/private subnet setup.

### CloudOps_Solution-Project

- **Goal:** Automate IAM user/group management for DevOps teams.
- **Skills:** Bash scripting, AWS IAM, group policies, automation.
- **Highlights:** Scripted user/group creation, policy attachment, admin group setup.

### GatoGrowFast_Project

- **Goal:** Manage AWS IAM for a marketing consultancy.
- **Skills:** IAM policies, user creation, permissions assignment.
- **Highlights:** Custom policies, user onboarding, screenshots.

### Linux_basic-commands_project2

- **Goal:** Practice essential Linux admin commands.
- **Skills:** File system, permissions, process management, sudo usage.
- **Highlights:** Real command outputs, screenshots, admin tips.

### linux_text_editor

- **Goal:** Learn Vim and Nano for Linux text editing.
- **Skills:** File editing, navigation, saving, exiting.
- **Highlights:** Stepwise guides, screenshots, editor comparison.

### linuxAdmin_project-1

- **Goal:** Intro to Linux admin on AWS EC2.
- **Skills:** EC2 management, SSH, instance operations, best practices.
- **Highlights:** Console access, instance control, MobaXterm usage.

### mini_project-git_basic

- **Goal:** Understand Git basics and collaboration.
- **Skills:** Git clone, branch, commit, push, merge.
- **Highlights:** Branching simulation, screenshots, HTML boilerplate.

### More_advance_linux_adminMini_project

- **Goal:** Advanced Linux admin: users, groups, permissions.
- **Skills:** User/group management, file permissions, system tasks.
- **Highlights:** Command tables, practical exercises, security tips.

### Security-group_NACL_mini-project

- **Goal:** Implement AWS Security Groups and NACLs with Terraform.
- **Skills:** VPC security, Terraform, inbound/outbound rules.
- **Highlights:** Stepwise setup, screenshots, rule configuration.

### Shell_script-Comments_project

- **Goal:** Master commenting in shell scripts.
- **Skills:** Single-line, multi-line, inline comments.
- **Highlights:** Commenting best practices, script examples.

### Shell_scripting-Capstone_Project

- **Goal:** Capstone Bash script for multiplication tables.
- **Skills:** Loops, input validation, error handling, modular scripting.
- **Highlights:** Full/partial tables, C-style loops, test scripts.

### Shell_scripting-control_flow

- **Goal:** Learn control flow in shell scripting.
- **Skills:** if-else, for/while loops, case statements.
- **Highlights:** Syntax guides, use cases, practical examples.

### Shell_scripting-functions_Project

- **Goal:** Modularize shell scripts with functions.
- **Skills:** Function definition, error handling, argument validation.
- **Highlights:** Case study, robust scripting, AWS CLI checks.

### Shell_scripting-handling_errors

- **Goal:** Handle errors in shell scripts, especially with AWS CLI.

### Shell_scripting-Mini_project

### Shell-scripting_for_cloud-computing
